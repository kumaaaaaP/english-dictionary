\(\angle \mathrm{PQR} = \theta\) とおくと、\(\theta\) の範囲は \(0 < \theta < \frac{\pi}{2}\) で、

\[ \begin{aligned} \frac{\mathrm{PQ} + a \mathrm{PR}}{\mathrm{QR}} &= \cos \theta + a \sin \theta \\ &= \sqrt{1+a^2} \sin (\theta + \alpha) \end{aligned} \]

ただし \(\alpha\) は下図のように定義する。

\(\theta\) は \(0 < \theta < \frac{\pi}{2}\) の範囲で動くから、最大値は \(\sqrt{1+a^2}\)

(3)

\(h(\theta) = \sqrt{1+a^2} \sin (\theta + \alpha)\) とおくと、 \[ h'(\theta) = \sqrt{1+a^2} \cos (\theta + \alpha) \]

(i) \(0 < \alpha \leqq \frac{\pi}{4}\) のとき

\[ h(0) < g(t) \leqq \sqrt{a^2+1} \] \(h(0) = 1, g(t) = 2-a\) より、 \[ 1 < 2-a \leqq \sqrt{a^2+1} \] これを解いて、 \(a < 1\) かつ \(a \geqq \frac{3}{4}\)

(ii) \(\frac{\pi}{4} < \alpha < \frac{\pi}{2}\) のとき

\[ h(\alpha) < g(t) \leqq \sqrt{a^2+1} \] \(h(\alpha) = a, g(t) = 2-a\) より、 \[ a < 2-a \leqq \sqrt{a^2+1} \] これを解いて、 \(a < 1\) かつ \(a \geqq \frac{3}{4}\)

(i), (ii) より、求める \(a\) の範囲は
\[ \frac{3}{4} \leqq a < 1 \]
